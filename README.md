# Container House Designer

Container House Designer je jednoduchá webová CAD aplikace pro návrh kontejnerových domů.

## Cíl

- 2D editor
- 3D zobrazení
- JSON jako jediný zdroj dat
- Automatické střechy
- Export OBJ / glTF
